Alugard-Drop is built for the modern web as the perfect successor to Dragula. It provides an intuitive, highly configurable API for drag-and-drop interactions while maintaining an ultra-lightweight footprint.

## Documentation

Full documentation, interactive examples, and API references are available here:

### Features
- **Framework Agnostic:** Works gracefully with Vanilla JS, React, Vue, Svelte, or any other library.
- **Ultra Lightweight:** Zero external dependencies. Keeps your bundle size incredibly small while delivering native-like physics.
- **First-Class TypeScript:** Provides comprehensive typings out of the box for an excellent developer experience and auto-completion.
